# 🌌 NebulaPC

NebulaPC is a cloud desktop web interface that lets users access a remote computer from their browser.

## 🚀 Features
- Mobile-friendly UI
- One-click desktop launch
- Works with Apache Guacamole backend

## ⚙️ Setup Backend
Follow these steps on your server:

1. Install Docker
2. Run Guacamole + Desktop containers
3. Replace YOUR-IP in script.js

## 🌐 Deploy
Upload this project to GitHub Pages or any hosting.
